gitalk搭建遇见的问题

blacklisten 2018-10-17 21:58:17 杂谈

HomePage URL 程序主页的完整地址即github仓库 eg：https:blacklisten.github.io github仓库地址必须是https 不可为http

Authorization callback URL Authorization callback URL：网站的callback地址 eg：https://blacklisten.cn callback URL 如果所绑定返回地址有SSL证书则为https 反之为http

如果为http的话 要去掉项目仓库中Github Page的Enforce HTTPS

# Error: Validation Failed

文章的评论区报Error: Validation Failed，原因是由于 Github 限制 labal 长度不能超过 50。Github的label默认是用encodeURI进行编码的，因此我们只需要用md5编码缩短label长度即可

具体步骤：
